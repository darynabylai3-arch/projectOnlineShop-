# projectOnlineShop-

🛍️ ShopMind — Online Store with AI Consultant
A full-stack e-commerce application built with Angular and Django REST Framework.

About
ShopMind is an online store where users can browse and purchase products. It includes an AI assistant that acts as a shopping consultant, helping users find what they need. Admins can manage the product catalog through a dedicated dashboard.

Features

🤖 AI Consultant — chat assistant that answers questions and recommends products
🛒 Shopping Cart — add items, adjust quantities, and place orders
🔧 Admin Panel — create and delete products
🔐 Authentication — user registration and login with role-based access


Roles

Admin — can create, edit, and delete products
Customer — can browse products, use the cart, and place orders


Tech Stack

Frontend: Angular
Backend: Django REST Framework
Database: PostgreSQL
Auth: JWT
